AULA 03 - O que é CSS?
Cascading Style Sheet

-> As Folhas deestilos em cascata (CSS) 
são um conjunto de regras de formatação que controlam a aparência de uma página da Web.

conceito básico = separar conteúdo da aparência

 HTML 		+ 	      CSS = website
Conteúdo 		    Aparência

por que estilo em cascata?
- Há várias formas de inserir modificações no estilo com CSS
- Ordem de prioridade
  ->Efeito cascata

EFEITO CASCATA

° folha deestilo padrão do navegador;			MENOR PRIORIDADE
° folha de estilo do usuário (se existir);			|
° folha de estilo do desenvolvedor;				|
	-estilo externo (importado). preferido				|
	-estilo incorporado (definido no head);			|
	-estilo inline(dentro de um elemento);			V
° Marcação !important;					MAIOR PRIORIDADE

Há três formas:
1. inline(style="")
2. tag <style></style>
3. importação/link

inline (maior prioridade)
-> feita diretamente na tag:
	<div style="height:100px"></div>
	<p style="color:red">Meu texto</p>
//somente esse parágrafo terar a cor vermelha

Tag <style>
-> Códigos são declarados diretamente no código HTML através da tag <style>
<head>
	<style>
		p{
			color: red;

		}
	</style>
</head>
<body>
	<p>Meu texto</p>
</body>

// nesse exemplo todos os parágrafos inseridos no texto terão a cor vermelha

Importação
-> Estilo importado de um arquivo ".css" separado
<head>
	<link rel="sylesheet" type="text/css" href="style.css(local do arquivo)" >
</head>
//maneira mais recomendada

seletor {

AULA 11 - Container

uma forma de definir um tamanho para a página é definindo o container
design responsivo bootstrap(pesquisar sobre)

container é uma div que vai segurar todo mundo

a div do container deve ser declarado com uma class, pois é desejável as vezes que se tenha mais de um container em uma página, logo se torna melhor que todos os containers possam ser "chamados" de uma vez só, por isso utilizamos uma class


declarando uma class

.container{
	width: 960px
	margin:0 auto
}
